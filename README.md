# 🔐 Генератор случайных паролей

Простая, но эффективная утилита для создания надежных паролей. 
Написана на Python без использования внешних библиотек.

## 📋 Описание

Программа генерирует случайные пароли с возможностью настройки:
- Длина пароля
- Использование цифр
- Использование специальных символов

Пароли создаются с использованием модулей `random` и `string`, что обеспечивает достаточную случайность для повседневного использования.

## 🚀 Как использовать

1. Убедитесь, что у вас установлен Python 3.6+
2. Скачайте файл `main.py`
3. Запустите программу:
   ```bash
   python main.py
